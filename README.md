# BitcoinTicker
Arduino code for my ESP32 based Bitcoin price ticker

# Connecting The Components On A Breadboard
| SSD1306 | ESP32 |
|-----|---|
| GND | GND |
| VCC | 3.3V |
| SLC | GPIO 22 |
| SDA | GPIO 21 |

Source - https://www.the-diy-life.com/bitcoin-ticker-using-an-esp32-and-oled-display/
